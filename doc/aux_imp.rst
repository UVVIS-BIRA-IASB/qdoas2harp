aux_imp
=======

qdoas to harp tool

.. code-block:: none

  Usage:
      aux_imp  [options] <qdoasfile> 
	      add extra variables to  <qdoasfile>
          Options:
                          -n perform a trial run and check which aux is needed
			  --molec directory where to put the output
			  --auxfile aux file from which variables are taken.
